# Superstore Sales & Profit Analysis Dashboard

## 📌 Project Overview
This project analyzes retail sales data (2014–2018) to identify key trends in sales, profitability, and customer segments.

## 🎯 Business Problem
The company is experiencing variation in sales and profit across regions, categories, and discount strategies. The goal is to identify underperforming areas and improve profitability.

## 📊 Tools Used
- Excel (Data Cleaning & EDA)
- Power BI (Dashboard)

## 🔍 Key Insights
- West region generated the highest sales
- Technology is the top-performing category
- High discounts (>30%) lead to losses
- Central region furniture category is loss-making

## 💡 Recommendations
- Limit discounts to 20%
- Focus on high-performing regions
- Improve pricing strategy for loss-making categories
